# Estimator - Refactored Project (Scenario 2)

This repository contains a refactored, component-driven version of your Estimator app.

Run a local static server (recommended `npx http-server` or `live-server`) and open `pages/index.html`.

Notes:
- Data files located in `assets/data/` (materials.json, tenaga.json, pekerjaan.json).
- JS modules are ESM; pages import them with `<script type="module">`.
